# Next Steps

- launchd / tunnel 설정값 정리
- 실서비스 재시작 절차 문서 보강

### 2. Mini App UX 마감
- 남은 영문 문구 정리
- 상태 탭 세부 문구 다듬기

- 모바일 터치 타깃과 밀도 추가 조정

### 3. 운영/지원성 강화

- 장애 재현 체크리스트 축적

### 4. 공개/릴리스 준비
